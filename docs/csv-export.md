# Експорт у CSV (`--csv`)

Запуск із прапорцем `--csv` додає рядки у файл у каталозі **`out/`** (каталог у репозиторії в `.gitignore`).

## Ім’я файлу

`out/<ім'я_бд>.csv` — те саме **базове ім’я**, що й у файлу SQLite **без** суфікса `.db` (наприклад `my_project` → `data/my_project.db` → `out/my_project.csv`).

## Що потрапляє у файл

- У файл **дописуються** лише рядки зі сканів, **успішно** оброблених у **поточному** запуску (після збереження в БД).
- Якщо в цьому запуску не було жодного такого скану — **файл не змінюється**.
- Заголовок колонок записується **лише** при створенні нового файлу або якщо файл **порожній** (перший допис).

## Колонки

| Колонка | Зміст |
|---|---|
| Папка | Відносна папка скану |
| Файл | Ім’я файлу |
| Скан | Номер скану (з імені файлу), якщо вдалось витягнути |
| Прізвище | |
| Ім'я | |
| Рік народження | |

Один рядок на **одну особу**; якщо для скану не знайдено жодної особи — один рядок із порожніми полями прізвища, імені та року.

## Приклад

```bash
python indexer.py my_project /шлях/до/сканів --csv
```

Детальніше про інші параметри — у [використанні](usage.md).
